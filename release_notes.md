Max Studio Hub now works on BOTH Windows and Mac.

**Downloads on this page:**
- **MaxStudioHub-Windows.zip** — for Windows 10 and newer. Unzip it and run MaxStudioHub.exe.
- **MaxStudioHub-macOS.zip** — for Macs (Apple chip and Intel, macOS 11+). Unzip it, drag "Max Studio Hub.app" into Applications, then allow it once in System Settings → Privacy & Security → "Open Anyway". A READ ME file with the exact steps is inside the zip.

**What is new:**
- Full Mac support — same app, same four tools (ComfyUI, Forge, Fooocus, Kohya).
- The app picks the right setup for your computer by itself: CUDA on Windows with NVIDIA cards, Apple Metal on Macs. No settings to change.
- Adding your own GitHub tools now works on Mac too (it understands run.sh / setup.sh projects).
- Help text inside the app matches your computer (Finder/Explorer, the right install tips).
- Every release is now tested and built automatically on real Windows and Mac machines before it reaches this page.
- The README now has simple step-by-step setup instructions for both systems.

Small note for Windows users: the download used to be called MaxStudioHub.zip — it is now MaxStudioHub-Windows.zip.
